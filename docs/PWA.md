# PWA (Progressive Web App)

Neon Occult NOLA is fully installable and works offline.

## Features
- **Manifest**: Located in `public/manifest.webmanifest` (generated by Vite PWA plugin).
- **Service Worker**: Handled by `vite-plugin-pwa`.
- **Offline Support**: Caches core assets and external fonts/images for offline viewing.
- **Theme Color**: #0f172a.

## Installation
The app can be installed on iOS, Android, and Desktop via the browser's "Add to Home Screen" or "Install" prompt.
